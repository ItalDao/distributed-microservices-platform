## Contributing

If you have contributed to this project and your name is not listed, please:

1. Submit a Pull Request to add yourself
2. Include your GitHub username and contribution area
3. Add a link to your GitHub profile

## How to Contribute

- Review [CONTRIBUTING.md](./CONTRIBUTING.md) for guidelines
- Check [Issues](https://github.com/ItalDao/distributed-microservices-platform/issues) for open tasks
- Submit Pull Requests with improvements
- Report bugs and suggest features

## Recognition Levels

### Core Contributors
Individuals who have made significant contributions to the project's core functionality.

### Documentation Contributors
Individuals who have improved project documentation, guides, and API specifications.

### Bug Reporters
Individuals who have reported and helped fix bugs.

### Feature Contributors
Individuals who have suggested and implemented new features.
